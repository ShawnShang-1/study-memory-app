# 软著材料说明

本目录用于存放当前项目的软件著作权申请辅助材料。

## 文件说明

- `software_manual.md`：根据当前 Flutter 工程实际实现整理的软件说明书初稿。
- `source_submission.txt`：由脚本自动汇总生成的核心 Dart 源代码交存文件。
- `source_manifest.md`：源代码交存统计信息与打印建议。

## 使用建议

1. 先确认你最终在申请表中使用的软件全称和版本号。
2. 如与当前默认值不同，请重新运行 `scripts/export_source_submission.ps1` 并同步修改 `software_manual.md` 中的名称和版本号。
3. 将 `software_manual.md` 导入 Word，添加页眉页脚并插入真实运行截图。
4. 将 `source_submission.txt` 导入 Word 或 WPS，设置等宽字体、行号、页眉页码后打印或导出 PDF。
